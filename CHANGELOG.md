## [0.1.0] — 2026-06-24

### Added
- 10 components: Button, Badge, Chip, Card, Select, Modal, Table, Hero, ClaraPanel, DraggableTicker
- React Aria accessibility layer across all interactive components
- CVA variant API with axes registry (variant, size, status, state)
- W3C DTCG token system — foundation + semantic layers, compiled via Style Dictionary
- Phase 3 agentic layer: Component Auditor, Design Review, A11y Agent, Docs Agent, Review PR
- `.ai/index.json` + `.ai/relationships/` — machine-readable component map for agents
- `.figma/maps/` — full variable and component ID mapping for Figma sync
- 22-node agentic system (see `AGENTIC-SYSTEM.md`)
- Lyse health audit integration (`npm run lyse`)
